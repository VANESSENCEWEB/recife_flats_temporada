# Fotos dos apartamentos

Coloque as fotos reais nesta pasta seguindo o padrão de nome em `scripts/data/apartamentos.js`:

```
boa-viagem-201-01.jpg
boa-viagem-201-02.jpg
pina-vista-01.jpg
espinheiro-01.jpg
boa-viagem-405-01.jpg
...
```

Enquanto as fotos locais não existirem, o site usa placeholders do Unsplash automaticamente.
